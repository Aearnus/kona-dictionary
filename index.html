Verbs

Basic Info

Verbs +-*%|&^!<>=~@?_,#$.:. Verbs work on all sensible types (ints, floats, strings, symbols, lists). Dyadic or monadic is determined from context, default is dyadic. Add : after a verb to force the monadic form, + is plus, +: is flip

Monadic

+
flip. transpose a matrix (a depth-2 list)

-
negate. invert sign

*
first. first element from the list

%
reciprocal. 1 over x

|
reverse. reverse order of list

&
where. &0 0 1 0 3 yields 2 4 4 4.

^
shape. #elts at each depth (min over)

!
enumerate. !4 yields 0 1 2 3

<
grade up. indices of list sorted ascending

>
grade down. indices of list sorted descending

=
group. =3 4 3 4 4 yields (0 2;1 3 4)

~
not. ~ 0 1 0 2 0 yields 1 0 1 0 1

@
atom. boolean is arg an atom (as opposed to a list)

?
unique. distinct elts from a list

_
floor. tolerant floor function

,
enlist. put x inside a 1-element list

#
count. number of elements in list

$
format. cast to string

.
execute/make dictionary. ."1+1" yields 2

Dyadic

+
plus. add numbers together

-
minus. subtraction

*
times. multiply two numbers

%
divide. x divided by y (not mod)

|
max/or. MAX(x,y) or OR(x,y)

&
min/and. MIN(x,y) or AND(x,y)

^
power. x to the exponent y

!
mod/rotate. 5!3 yields 2; 1 ! 4 5 6 yields 5 6 4

<
less. boolean is x less than y

>
more. boolean is x greater than y

=
equals. 1 2 3 = 1 2 4 yields 1 1 0 (tolerantly)

~
match. 1 2 ~ 1 2 yields 1 (types must match)

@
at. elts from x at indices y

?
draw. n?m yields n numbers < m

?
deal. -n?m yields n ints < m without replacement

?
sample. n?y yields n random elements from list y (-n for no replacement)

?
find. x?y yields index of y in list x (or #x)

?
invert. {x^2} ? 2 yields sqrt(2)

_
drop/cut. lose x elts from list y / separate into pieces

,
join. "ab","cd" yields "abcd"

#
take/reshape. fill x elts from y

$
form/format. cast

.
value. 1 2 3 . ,2 yields 3. see \.

Triadic

@
monadic amend. see \.

?
invert-guess. secant method with clues ?[{2^x};17;4]

.
monadic amend. see \.

Tetradic

@
dyadic amend. see \.

.
dyadic amend. see \.

Adverbs

Basic Info

Adverbs / \ ' /: \: ':. Use adverbs in conjunction with verbs/functions. Over and scan behave differently depending on whether their left argument. is monadic or dyadic.

/
over_dyad
+/1 2 3 or 4+/1 2 3 */1+!5

\
scan_dyad
+\3 4 5 or 6+\3 4 5 *\10#2

/
over_monad
apply until fixed f/x (%[;2.0])/9999.0

/
over_monad
apply n times n f/x 4 (2+)/0

/
over_monad
apply while true b f/x {x<10} {x+1}/ 3

\
scan_monad
trace until repeat f\x (1!)\1 2 3 4

\
scan_monad
trace n times n f\x 10(|+\)\1 1

/
over
{x+y+z}/[1 2 3;4;7 8 9] f/[x;y;z]

\
scan
{x+y+z}[1 2 3;4;7 8 9] f\[x;y;z]

/
join
","/("a";"b")

\
split
","\"a,b"

'
each
"abc" ,' "def" join each

'
each
!:' 2 3 4 enumerate each

/:
eachright
#:/:(2;2 2;2 2 2) count each right

/:
eachright
2 #/: ("alpha";"bravo";"charlie") take each right

\:
eachleft
0 1 2 3 #\: "abc" take each left
